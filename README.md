## 个人留言
不推荐你用YinwuChat实现跨服聊天，这个插件缺乏积极维护。我推荐你试试TrChat，配置真的非常简单，屏蔽词更高效，并且支持多频道消息。（你甚至可以用TrChat实现YinwuChat 95%的功能）

如果你因为需要网页聊天等原因一定要用它，记得安装InteractiveChat修补颜色代码问题，并禁用跨服广播死亡消息。
