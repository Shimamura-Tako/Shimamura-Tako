## 个人留言
总之，不要用YinwuChat，我推荐你试试TrChat，配置真的非常简单，屏蔽词更高效，并且支持多频道消息。

如果你因为需要网页聊天等原因一定要用它，记得安装InteractiveChat修补颜色代码问题，并禁用跨服广播死亡消息。

PonziScheme更是大份，这东西完全没有存在的必要，小服没效果大服用不到。
